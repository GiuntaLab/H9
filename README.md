# H9
The reference genome for the human embryonic stem cell H9

<p align="justify"> This repository is dedicated to sharing the T2T diploid assemblies of the H9 (WAe009-A) laboratory cell line, which is the most employed and currently available human stem cell for research purposes. It has been registered in the European and NIH (National Institute of Health) registries, and validated for its use in experimental settings worldwide. </p>[>
