# <p align="center">Requirements to identify structural rearrangements<p>
<p align="justify">
  Structural rearrangements have been primarily determined by comparing the two H9 haplotypes as well as each one of them, independently, against the <a href=https://www.ncbi.nlm.nih.gov/datasets/genome/GCF_009914755.1/">CHM13</a> reference genome. All these analyses were done using the <a href="https://github.com/schneebergerlab/syri"><code>SyRI</code></a> tool; they encompass the two main panles of Figure 5A and B.
</p>
<p align="justify">
  The exact commands used for both analyses are reported in ..., while the plot output was done with a custom script for Figure 5A and the built-in <a href="https://github.com/schneebergerlab/plotsr"><code>plotsr</code></a> function of <code>SyRI</code>. <i>Below</i> is also the code to compute: SNVs desity in chromosome arms and centromeres separately, the total lenght of INDELs as reported by <code>SyRI</code>, and the total length of other complex SVs. To calculate the average unaligned sequence block length it was employed a combination of <a href="https://github.com/samtools/samtools"><code>samtools</code></a> and <a href="https://www.gnu.org/software/datamash/"><code>GNU datamash</code></a> on the <code>SyRI</code> output BAM of each H9 haplotype aligned against CHM13, the command is similarly reported in the code-block to follow. Other specific metrics and values have been derived directly from the <code>SyRI</code> output files.
</p>

```sh
#gets centromeres only SNPs density
bedtools intersect -a H9hap2_hap1.snp.density -b H9v1.0.hap1.three.bed | awk 'BEGIN{FS=OFS="\t"} {if ($4>0) print $0}' > H9hap2_hap1.snp.density.cent
#gets arms only SNPs density
bedtools subtract -a H9hap2_hap1.snp.density -b H9v1.0.hap1.three.bed | awk 'BEGIN{FS=OFS="\t"} {if ($4>0) print $0}' > H9hap2_hap1.snp.density.arms

#total length of INSertions and DELetions by SyRI
grep "INS" <syri_genome_alignment>.out | awk 'BEGIN{FS=OFS="\t"} {print $7,$8,$3=$8-$7}' | awk 'BEGIN{FS=OFS="\t"} {if ($3>49 && $3<501) print $0}' | awk 'BEGIN{FS=OFS="\t"} { sum += $3 } END { print sum }'
grep "DEL" <syri_genome_alignment>.out | awk 'BEGIN{FS=OFS="\t"} {print $1,$2,$3=$2-$1}' | awk 'BEGIN{FS=OFS="\t"} {if ($3>49 && $3<501) print $0}' | awk 'BEGIN{FS=OFS="\t"} { sum += $3 } END { print sum }'

#total length of other complex SVs by SyRI
grep -vP "INS|DEL" <syri_genome_alignment>.out | awk 'BEGIN{FS=OFS="\t"} {print $7,$8,$3=$8-$7}' | awk 'BEGIN{FS=OFS="\t"} {if ($3>100000) print $0}' | awk 'BEGIN{FS=OFS="\t"} { sum += $3 } END { print sum }'
###variants excluding synteny, not aligned, syntenic not alignment and HDRs
grep -vP "INS|DEL|SYN|NOTAL|SYNAL|HDR" <syri_genome_alignment>.out | awk 'BEGIN{FS=OFS="\t"} {print $7,$8,$3=$8-$7}' | awk 'BEGIN{FS=OFS="\t"} {if ($3>100000) print $0}' | awk 'BEGIN{FS=OFS="\t"} { sum += $3 } END { print sum }'

#to be computed for each BAM of H9 on CHM13 as generated by SyRI and, then, averaged
samtools view <BAM> | awk '{ print length($10); }' | datamash mean 1
```

<p align="justify">
where the <i>H9hap2_hap1.snp.density</i> and the <i>H9v1.0.hap1.three.bed</i> in the first part of the code-block were genereted based on <a href="https://github.com/GiuntaLab/RPE1/blob/main/RPE1v1.1/Figure_3/scripts/snp_density_hdr.py">this script</a> available on the RPE1 repository and the steps documented in the <a href="https://github.com/GiuntaLab/H9/tree/main/Centromere%20annotation">Centromere annotation</a> section of this repository, respectively.
</p>
