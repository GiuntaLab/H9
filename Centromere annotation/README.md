# <p align="center">Requirements to run centromeres annotations<p>
<p align="justify">
  Centromeres annotations have been produced using three independent, orthogonal approaches to confirm and validate the actual results. In order, were used the following tools:</br>
  
  1. <a href="https://github.com/lh3/dna-nn"><code>dna-nn</code></a> — annotation in the SNVs density plot of main Figure 5A,</br>
  2. <a href="https://github.com/fedorrik/HumAS-HMMER_for_AnVIL"><code>HumAS-HMMER_for_AnVIL</code></a> — annotation to compare centromeres' lenght between RPE1 and H9, and</br>
  3. <a href="https://github.com/GiuntaLab/GCP-Centeny"><code>GCP-Centeny</code></a> — annotation of CENP-B box density and orientation (<i>centeny map</i>) of main Figure 3 panel A and B
</p>

<p align="justify">All three tools were executed with default parameters, and the exact command(s) are listed on the respective repositories linked above. Aside from that, some post-processing was necessary for the output of <a href="https://github.com/lh3/dna-nn"><code>dna-nn</code></a> to select only α-satellite regions as follow:</p>

```sh
awk '$4==2' H9v1.0.hap1.bed > H9v1.0.hap1.three.bed #where the H9v1.0.hap1.bed is the rich BED generated by dna-nn
```

<p align="justify">As you can see, this was computed for haplotype 1 only since the SNVs density is the result of a genome-to-genome alignment obtained from a <a href="https://github.com/schneebergerlab/syri"><code>SyRI</code></a> output with haplotype 1 as reference and haplotype 2 as query. The <a href="https://github.com/bernatgel/karyoploteR"><code>karyoploteR</code></a> of Figure 5A displays, indeed, haplotype 1 chromosome coordinates.</p>
